## HOW TO USE:

1. **Generate images** using ChatGPT, Midjourney, or any AI tool with the prompts above
2. **Download** each image
3. **Rename** to the exact filename shown (e.g., `salicylic-cleanser.png`)
4. **Upload** to: `d:\Layl Website\layl-website\public\products\`
5. **Run this command** to update database:
   ```bash
   node update-all-images.js
   ```

---

## TIPS FOR BEST RESULTS:

- Use **1024x1024** or **1024x1536** (portrait) resolution
- Ensure **pure white background** (#FFFFFF)
- Keep **LAYL branding consistent** (same font style across all products)
- **Center the product** in frame
- Use **professional studio lighting**
- Maintain **minimalist clinical aesthetic**

---

## Quick Alternative (If generating is too time-consuming):

You can also use **Canva** free templates:
1. Search "Product Mockup" on Canva
2. Use a white background template
3. Add text: "LAYL" + product name
4. Export as PNG
5. Upload to `public/products/`
